About me

I'm William, a student studying Software Engineering at the University of Waterloo. I've worked with a variety of different technologies, including from web development, mobile, IoT, and robotics. By broadening my skills in different fields, I hope to gain new perspectives when tackling different projects in the future.

Experience

Agile Software Engineer at TribalScale

I worked on multiple platforms, ranging from web, mobile, and IoT. I developed TribalScale and Athos' new websites and used with server-side React.js for faster loading. Afterwards I worked on a CI/CD pipeline for automatic deployment of various types of applications, and an internal Android app. Finally, I refactored and developed the PGA Tour App, a cross-platform golf app for Google Home and Amazon Alexa, where I improved the name recognition and added tests and support for new game modes.

Web App Developer at Innovasium Digital

Developed AdvanceX, a web app that builds other web applications and portals. I used React.js for the front-end to develop features such as a rich text editor with Draft.js, graphs with HighCharts.js, forms, and file management systems. I led the team to refactor most of the application infrastructure to work with various Rails API calls. I created a smoother user interface by optimizing the drag and drop functions to run from linear time to constant time. It was a great learning experience for agile web development in team setting.

Software Engineering Intern at Quanser

In grade 11, my high school's robotics team's sponsor, Quanser, gave me an offer to be an intern at their company to learn more about robotics and control systems. During my time there, I tested and built features for Arduino, Raspberry Pi, and myRIO systems. I also developed software that controlled an inverse pendulum with a Microsoft Kinect, which was featured at the company's academic keynote in NI Week 2014.
